须知： 微信公众号申请要过很多审核，想最终成功，还需要把开发的网站申请域名
      并添加到公众号中进行匹配。还要开通很多接口。过程还是很艰辛的。

      不过，测试阶段可以申请测试号，不过测试号是没有微信扫码支付等接口的。
      测试号的使用也是要配域名的，不过测试号上可以配假域名，若想保证可以测通，一般还需要内网穿刺命令。
      这个命令工具中要配置相同域名。启动后，才可保证使用。
      我当时测试的时候，内网穿刺用的是这个ngrok

此套代码测试号和正式号都可以使用
1、把WeiXinUtil中的appid和appsecret改了
2、将Controller中的public Map<String,Object> weixin（）方法里的
    String url 中的redirect_uri和appid改成你的。
3、需要注意的是，代码跑通后，如果是正式号，必须要在上线环境下才可测试成功。
    因为redirect_uri是扫码时要访问的方法，而此方法又必须配置域名，所以。。。
4、但如果你是测试号，好像可以直接跑通，测试号开发太久远了，我已经记不清了。
